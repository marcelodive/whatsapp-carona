# whatsapp-carona

Bot para organizar caronas.

## Como instalar?

- Renomeie o arquivo `globals.ts.example` para `globals.ts`
  - Adicione sua chave do Google Maps API em `MAPS_KEY`.
  - Adicione o CEP de destino em `DESTINY_CEP`.
- Execute `npm run start`.
