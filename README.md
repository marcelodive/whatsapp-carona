# whatsapp-carona

Bot to organize carpooling.

## How to Install?

- Rename the file `globals.ts.example` to `globals.ts`.
  - Add your Google Maps API key in `MAPS_KEY`.
  - Add the destination ZIP code in `DESTINY_CEP`.
- Run `npm run start`.
